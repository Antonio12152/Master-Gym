3. Ist-/ Soll Analyse 

Ist-Analyse 

Die Ist-Analyse beschreibt den Ausgangszustand eines Projekts, sprich die akute Sachlage. 

    • Keine Webseite → Keine gute Auffindbarkeit für „Master Gym“ 

    • Plannen wird als Bild gepostet → unpraktisch, nicht barrierefrei, nicht suchbar 

    • Gibt es Bilder? Gibt es eine CI (Corporate Identity)? Gibt es Texte?
